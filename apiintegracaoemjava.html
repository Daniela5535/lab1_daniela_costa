O que é REST API com Java?

  • Voltar ao Início

O Básico

REST API é uma forma de programas conversarem pela internet. É como pedir informações ou enviar dados usando comandos simples, como "pegar" (GET), "criar" (POST), "mudar" (PUT) ou "apagar" (DELETE).

Em Java, você pode fazer duas coisas: criar um servidor que responde a essas pedidos (back-end) ou fazer um programa que pede informações a um servidor (front-end).

Como Funciona

  • Recursos: Coisas como usuários ou produtos, com endereços como /users.
  • Comandos: GET para ver, POST para adicionar, etc.
  • Sem Memória: Cada pedido é independente.
  • Formato: Geralmente usa JSON, que é texto simples.

Fazendo um Servidor (Back-End)

Use Spring Boot para criar um servidor simples. Ele responde a pedidos da internet.

Exemplo Fácil

// Código para o servidor
@RestController
@RequestMapping("/users")
public class UserController {
    @GetMapping
    public List getUsers() {
        return List.of("João", "Maria"); // Lista simples
    }
    
    @PostMapping
    public String addUser(@RequestBody String name) {
        return "Utilizador" + name + " adicionado!";
    }
}

Isso cria um lugar onde você pode pedir a lista de usuários ou adicionar um novo. Teste abrindo no navegador: http://localhost:8080/users.

Pedindo Informações (Front-End)

Em um programa Java, use HttpClient para falar com o servidor.

Exemplo Fácil

// Código para pedir dados
HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("http://localhost:8080/users"))
    .GET()
    .build();
HttpResponse response = client.send(request, HttpResponse.BodyHandlers.ofString());
System.out.println("Resposta: " + response.body()); // Mostra a lista

Isso pede a lista de utilizadores e mostra na tela. É como fazer uma pergunta e ouvir a resposta.

Dicas Simples

  • Use HTTPS para proteger os dados.
  • Teste com ferramentas como Postman.
  • Comece pequeno e adicione mais depois.
