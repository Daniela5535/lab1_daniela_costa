Programação na web

Na Escola do Comércio de Lisboa, na disciplina de Tecnologia 1 no 3ºBTIS, iremos aprender Programação web 😅 e abordamos os principios básicos associados ao desenvolvimento para a Web, apenas do lado front-end, lado visual. Também sabmos que existe um lado back-end, que é o lado do servidor, mas nao iremos abordar esse lado.

Aprender-se-ão as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • JavaScript

De forma a consolidar todos os conhecimentos, serão feitos alguns laboratórios.
